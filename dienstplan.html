README – Einrichtung in 5 Minuten
1. Firebase-Konto erstellen
→ console.firebase.google.com → "Projekt erstellen" (kostenlos)

2. Realtime Database anlegen
→ Im Projekt: "Realtime Database" → "Datenbank erstellen"
→ Standort: Europe (belgium) → Testmodus (30 Tage offen)

3. URL kopieren
→ Sieht so aus: https://dein-projekt-default-rtdb.europe-west1.firebasedatabase.app

4. Regeln dauerhaft öffnen (optional nach 30 Tagen)
→ Regeln-Tab → ersetze durch:
{"rules": {".read": true, ".write": true}}
5. Website hosten (GitHub Pages)
→ github.com → "New repository" → Public → Datei hochladen
→ Settings → Pages → Branch: main → Speichern
→ Deine URL: https://DEINNAME.github.io/REPO-NAME/dienstplan.html

→ Link in die WhatsApp-Gruppe schicken, fertig! 🎉
